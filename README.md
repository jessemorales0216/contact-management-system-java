# Contact Management System

## Overview

This project is a Java-based contact management application that demonstrates object-oriented programming, software testing, and quality assurance principles.

The application allows users to create, retrieve, update, and delete contacts while validating user input to maintain data integrity. Automated unit tests were developed using JUnit to verify functionality and ensure expected behavior.

## Technologies

- Java
- JUnit
- Git
- Object-Oriented Programming (OOP)

## Features

- Create contacts
- Update contact information
- Delete contacts
- Retrieve contacts by ID
- Input validation
- Duplicate contact ID prevention
- Exception handling
- Automated unit testing

## Skills Demonstrated

- Object-Oriented Programming
- CRUD Operations
- Unit Testing
- Input Validation
- Exception Handling
- Service Layer Design
- Software Quality Assurance
